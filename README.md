# 💻 Single Language Projects

A collection of programming projects built using individual programming languages to strengthen problem-solving, programming fundamentals, and software development skills.

## 📌 About

This repository contains projects developed using a single programming language without relying on external frameworks. The goal is to build a strong foundation in programming concepts, data structures, algorithms, file handling, and project organization.

## 🛠️ Languages & Projects

### C

* Student Management System
* Concepts: Structures, Functions, File Handling, Searching

### C++

* Library Management System
* Concepts: Object-Oriented Programming, Classes, Inheritance, STL

### JavaScript

* Expense Tracker
* Concepts: DOM Manipulation, Local Storage, Event Handling

## 🚀 Skills Demonstrated
